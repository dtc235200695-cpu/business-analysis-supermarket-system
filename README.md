# 🛒 Business Analysis – Supermarket Management System

## 📌 Overview
This project focuses on analyzing and designing a Supermarket Management System from a Business Analyst perspective. The project includes requirement gathering, system modeling, and documentation.

## 🎯 Objectives
- Understand business requirements
- Model system using UML
- Create professional BA documents (URD, SRS)
- Manage project timeline

## 📂 Project Artifacts
- 📄 URD (User Requirement Document)
- 📄 SRS (Software Requirement Specification)
- 📊 Gantt Chart
- 🧩 UML Diagrams (Use Case, Class, Business Process)

## 🛠 Tools & Technologies
- Draw.io (UML diagrams)
- Microsoft Excel (Gantt chart)
- Word (Documentation)
- Agile/Scrum (basic understanding)
